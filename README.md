Submitted by CHITHRAPRIYA S

---

## 🧛‍♀️ About the Project

This project is a responsive multi-page website designed for an exclusive **LBTITW Vampire Diaries 80’s Theme Party**.

The website captures the dark, mystical vibe of *The Vampire Diaries* universe with a **bloody red and black aesthetic**, inspired by characters like Katherine, Stefan, Damon, and more.

It allows students to:
- Explore the event
- Get outfit inspiration
- Register for entry (local version with database)

---

## 🌙 Features

- 🔥 Dark vampire-themed UI (red + black aesthetic)
- 📱 Fully responsive (mobile + desktop)
- 🧾 Registration page for event entry
- 👗 Outfit inspiration section with character references
- 🐺 Includes Vampire, Werewolf, Witch themes
- 🌲 Mystic Falls inspired forest background
- ✨ Animations (blood effects, glow, hover effects)
